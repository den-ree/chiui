# DiaryApp Example

This app demonstrates how to use Chiui in a real SwiftUI flow, including store updates, local state, and contextual bindings.

## Run

1. Open `Examples/DiaryApp/DiaryApp.xcodeproj` in Xcode.
2. Select the `DiaryApp` scheme.
3. Run on iOS Simulator or macOS target.

## What to Explore

- Main list flow in `DiaryApp/Flows/Main/Screens/DiaryList`.
- Entry editing flow in `DiaryApp/Flows/Main/Screens/DiaryEntry`.
- Context/store wiring in `DiaryApp/Flows/Main/Store`.
